#PolicyNav – Public Policy Navigation Using AI
Infosys Springboard Virtual Internship 6.0 – Batch 13

#Project Overview
PolicyNav is an AI-powered system designed to simplify the navigation, understanding, and analysis of complex public policy and regulatory documents. The project focuses on leveraging Artificial Intelligence and Natural Language Processing (NLP) techniques to make public policies more accessible to policymakers, analysts, organizations, and citizens.

This repository is part of the Infosys Springboard Virtual Internship 6.0 (Batch 13) and demonstrates practical applications of AI in the public policy domain.
